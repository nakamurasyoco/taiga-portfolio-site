# Portfolio Site

このフォルダは、作品追加がしやすい構成のシンプルなポートフォリオサイトです。

## 使い方

1. `assets/images` に画像を追加
2. `data/projects.js` の配列に作品情報を1件追加
3. `index.html` をブラウザで開く

## 作品追加テンプレート

`data/projects.js` の `window.PROJECTS = [ ... ]` の中に次の形式で追加してください。

```json
{
  "title": "作品タイトル",
  "image": "assets/images/project-07.png",
  "category": "Web Design",
  "description": "作品の説明",
  "year": "2026"
}
```
